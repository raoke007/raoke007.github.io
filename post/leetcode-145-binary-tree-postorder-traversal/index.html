Given a binary tree, return the postorder traversal of its nodes' values.

Example:
	Input: [1,null,2,3]
   1
    \
     2
    /
   3
	Output: [3, 2, 1]

思路：

  • 返回类型为Integer的集合，存储链表每个节点中的值
  • 要求前序遍历，则按照左孩子、右孩子、父节点的顺序进行遍历操作
  • 递归函数要传入一个List，递归终止条件为node == null

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> list = new ArrayList<>();
        
        return postorderTraversal(root, list);
    }
    
    private List<Integer> postorderTraversal(TreeNode node, List<Integer> list) {
        if (node == null) {
            return list;
        }
        
        list = postorderTraversal(node.left, list);
        list = postorderTraversal(node.right, list);
        list.add(node.val);
        
        return list;
    }
}
