Given a string containing digits from 2-9 inclusive, return all possible letter combinations that the number could represent.

A mapping of digit to letters (just like on the telephone buttons) is given below. Note that 1 does not map to any letters.

Example:
  Input: "23"
  Output: ["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"].

Note:
  Although the above answer is in lexicographical order, your answer could be in any order you want.

暴力实现法

分析：

  • 依题意要获得所有组合情况，可以当做树形问题，如下图所示

  • 按照上图所示，获取所有字符串组合的过程就是箭头走向的过程，通过搜索整棵树，来查找这个问题的解，这种解决问题的思想就是回溯法

  • 回溯是一种算法思想，本质其实是在树形的结构中进行穷举

  • 回溯法通常是暴力解法的实现方式的一种

思路：

  • 每个数字都对应一个字符串，此时可以使用映射或哈希表来建立这种对应关系，此处使用哈希表（数组下标和字符串的对应关系）
  • 对于给定的“23”字符串转换为树形结构后，使用递归来实现，递归发生的地方在要想获得从“2”开始的数字字符串所有的字符串组合，只需要获得“2”这个字符之后的数字字符串所有的字符串组合，再加上“2”这个字符代表的所有字符串（这里就是a、b、c），最后得出的字符串组合就为“23”这个数字字符串能组合的所有字符串

class Solution {
    private final String[] letters = {"abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};

    private List<String> ret = new ArrayList<>();

    public List<String> letterCombinations(String digits) {
        if (!"".equals(digits) && !" ".equals(digits)) {
            findCombination(digits, 0, "");     
        }

        return ret;
    }

    private void findCombination(String digits, int index, String s) {
        // 只要字符串长度等于数字字符串长度时，就可以往回进行查找
        if (s.length() == digits.length()) {
            ret.add(s);
            return;
        }

        char c = digits.charAt(index);
        String letter = letters[c - '2'];
        for (int i = 0; i < letter.length(); i++) {
            // 进行下一次递归，进行下一个数字字符的所有字符串组合查找动作
            findCombination(digits, index + 1, s + letter.charAt(i));
        }
    }
}
