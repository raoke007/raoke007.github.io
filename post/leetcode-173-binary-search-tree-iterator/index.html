Implement an iterator over a binary search tree (BST). Your iterator will be initialized with the root node of a BST.

Calling next() will return the next smallest number in the BST.

Example:
   7
  /  \
  3   15
     /  \
    9    20

BSTIterator iterator = new BSTIterator(root);
iterator.next();    // return 3
iterator.next();    // return 7
iterator.hasNext(); // return true
iterator.next();    // return 9
iterator.hasNext(); // return true
iterator.next();    // return 15
iterator.hasNext(); // return true
iterator.next();    // return 20
iterator.hasNext(); // return false

Note:
	1. next() and hasNext() should run in average O(1) time and uses O(h) memory, where h is the height of the tree.
	2. You may assume that next() call will always be valid, that is, there will be at least a next smallest number in the BST when next() is called.

实现

暴力实现法

❗️其实工作中遇到的大部分问题，第一目标永远应该是先实现（不考虑时间、空间复杂度、不考虑代码质量等），再考虑优化的问题。

思路：

​ 前提：

  • 由题意每次调用next()后，返回二分搜索树中最小的值。第一想法是在next()中先查询最小节点的值，然后删除二分搜索树的最小元素节点
  • 查询最小节点元素值逻辑：依据二分搜索树定义，由根节点递归节点左孩子，则叶子节点的值就是最小值。
  • 删除最小元素节点逻辑：由根节点递归节点左子树，找到最后一个左孩子为空的节点，判断此节点如果有右孩子，则把此节点的右子树连接到此节点的父节点的左孩子上。

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class BSTIterator {

    private TreeNode root;

    public BSTIterator(TreeNode root) {
        this.root = root;
    }

    private int removeMin() {
        // 1. find minimum
        int ret = minimum();
        // 2. remove minimum
        root = removeMin(root);

        return ret;
    }

    private int minimum() {
        return minimum(root);
    }

    private int minimum(TreeNode node) {
        if (node.left == null) {
            return node.val;
        }

        return minimum(node.left);
    }

    private TreeNode removeMin(TreeNode node) {
        if (node.left == null) {
            TreeNode rightNode = node.right;
            node.right = null;

            return rightNode;
        }

        node.left = removeMin(node.left);

        return node;
    }

    /**
     * @return the next smallest number
     */
    public int next() {
        return removeMin();
    }

    /**
     * @return whether we have a next smallest number
     */
    public boolean hasNext() {
        return root != null;
    }
}

/**
 * Your BSTIterator object will be instantiated and called as such:
 * BSTIterator obj = new BSTIterator(root);
 * int param_1 = obj.next();
 * boolean param_2 = obj.hasNext();
 */

优化

前提：

思路：

  • 根据描述Note说明，next()和hasNext()平均时间复杂度为O(1)，第一种解法过于复杂，每次查询最小值和删除最小值平均时间复杂度为O(h)
  • 重新审题后，发现要求只要每次返回最小值即可，这时想到二分搜索树的中序遍历就可以把元素按从小到大顺序输出，这样只需要用队列接收中序遍历的元素即可

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class BSTIterator {

    private Queue<Integer> queue = new LinkedList<>();

    public BSTIterator(TreeNode root) {
        inOrder(root);
    }

    // 中序遍历，使用队列保存元素
    private void inOrder(TreeNode node) {
        if (node == null) {
            return;
        }

        inOrder(node.left);
        queue.add(node.val);
        inOrder(node.right);
    }

    /**
     * @return the next smallest number
     */
    public int next() {
        return queue.poll();
    }

    /**
     * @return whether we have a next smallest number
     */
    public boolean hasNext() {
        return !queue.isEmpty();
    }
}

 * boolean param_2 = obj.hasNext();
