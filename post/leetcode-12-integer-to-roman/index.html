Roman numerals are represented by seven different symbols: I, V, X, L, C, D and M.

Symbol       Value
I             1
V             5
X             10
L             50
C             100
D             500
M             1000

For example, two is written as II in Roman numeral, just two one's added together. Twelve is written as, XII, which is simply X + II. The number twenty seven is written as XXVII, which is XX + V + II.

Roman numerals are usually written largest to smallest from left to right. However, the numeral for four is not IIII. Instead, the number four is written as IV. Because the one is before the five we subtract it making four. The same principle applies to the number nine, which is written as IX. There are six instances where subtraction is used:
  - I can be placed before V (5) and X (10) to make 4 and 9. 
  - X can be placed before L (50) and C (100) to make 40 and 90. 
  - C can be placed before D (500) and M (1000) to make 400 and 900.

Given an integer, convert it to a roman numeral. Input is guaranteed to be within the range from 1 to 3999.

Example 1:
  Input: 3
  Output: "III"
Example 2:
  Input: 4
  Output: "IV"
Example 3:
  Input: 9
  Output: "IX"
Example 4:
  Input: 58
  Output: "LVIII"
  Explanation: L = 50, V = 5, III = 3.
Example 5:
  Input: 1994
  Output: "MCMXCIV"
  Explanation: M = 1000, CM = 900, XC = 90 and IV = 4.

暴力实现法

思路：

  • 使用映射map保存罗马数字和对应阿拉伯数字[1-I, 2-II, 3-III, 4-IV, 5-V, 9-IX, 10-X, 40-XL, 50-L, 90-XC, 100-C, 400-CD, 500-D, 900-CM, 1000-M]，把2，3，4放进去是因为当num在下述循环操作中最后会处于[1,5]区间中，这里直接存放对应罗马数字即可

  • 使用数组nums保存罗马数字对应数值 [1, 2, 3, 4, 5, 9, 10, 40, 50, 90, 100, 400, 500, 900, 1000]，同理把2，3，4放进去是因为当num在下述循环操作中最后会处于[1,5]区间中

  • 对于给定num，初始化返回结果字符串 res，每次循环，在nums中找到第一个比num大的元素，对它前一个元素进行求余(/)，根据求余结果计算对应map中对应的罗马数字，再对它进行求模(%)，求模结果如果直接在map中可以找到，则直接返回res+map.get(求模结果)，同时循环终止，否则把求模结果赋值给num，进入下一次循环

举例：对于给定 num = 364

  1. 在nums中找到第一个大于364的元素（此时为400），求余 364/ 100 = 3，此时 res = “CCC”, 求模 364 % 100 = 64，64不在map中，则 num = 64，进入下一次循环
  2. 在nums中找到第一个大于64的元素（此时为90），求余 64/ 50 = 1，此时 res = “CCCL”, 求模 64 % 50 = 14，14不在map中，则 num = 14，进入下一次循环
  3. 在nums中找到第一个大于14的元素（此时为40），求余 14/ 10 = 1，此时 res = “CCCLX”, 求模 14 % 10 = 4，4在map中，则返回“CCCLX” + map.get(4) = “CCCLXIV”，终止循环

class Solution {
    public String intToRoman(int num) {
        Map<Integer, String> map = new HashMap<>();
        map.put(1, "I");
        map.put(2, "II");
        map.put(3, "III");
        map.put(4, "IV");
        map.put(5, "V");
        map.put(9, "IX");
        map.put(10, "X");
        map.put(40, "XL");
        map.put(50, "L");
        map.put(90, "XC");
        map.put(100, "C");
        map.put(400, "CD");
        map.put(500, "D");
        map.put(900, "CM");
        map.put(1000, "M");
        
        if (map.containsKey(num)) {
            return map.get(num);
        } else {
            int[] nums = {1, 2, 3, 4, 5, 9, 10, 40, 50, 90, 100, 400, 500, 900, 1000};
            int size = nums.length;
            
            StringBuilder sb = new StringBuilder();
            boolean flag = true;
            int index = size;
            while (flag) {
                for (int i = 0; i < size; i++) {
                    if (nums[i] > num) {
                        index = i;
                        break;
                    }
                }

                int times = num / nums[index - 1];
                if (times > 1) {
                    for (int j = 0; j < times; j++) {
                        sb.append(map.get(nums[index - 1]));
                    }
                } else {
                    sb.append(map.get(nums[index - 1]));
                }

                int modulus = num % nums[index - 1];
                if (map.containsKey(modulus)) {
                    sb.append(map.get(modulus));
                    flag = false;
                } else {
                    if (modulus == 0) {
                        flag = false;
                    } else {
                        num = modulus;
                    }
                }
            }
            
            return sb.toString();
        }
    }
}
