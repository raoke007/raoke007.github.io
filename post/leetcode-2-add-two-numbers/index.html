You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Example:
  Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
  Output: 7 -> 0 -> 8
  Explanation: 342 + 465 = 807.

实现

暴力实现法：审题后脑子里能想到的第一种实现方法

思路：

  • 分别将链表中的数据反向取出来，这时想到可以用栈存储数据
  • 分别使用StringBuilder接收出栈的数据
  • 分别转化为int型，执行+操作
  • 转换为字符串然后反转
  • 把结果字符串转换为结果链表，返回

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {        
        // 1. 分别使用栈存储l1, l2数据
        Stack<Integer> stack1 = new Stack<>();
        while (l1 != null) {
            stack1.push(l1.val);
            l1 = l1.next;
        }
        Stack<Integer> stack2 = new Stack<>();
        while (l2 != null) {
            stack2.push(l2.val);
            l2 = l2.next;
        }
        // 2. 分别使用StringBuilder接收出栈的数据
        StringBuilder sb1 = new StringBuilder();
        while (!stack1.isEmpty()) {
            sb1.append(stack1.pop());
        }
        StringBuilder sb2 = new StringBuilder();
        while (!stack2.isEmpty()) {
            sb2.append(stack2.pop());
        }
        // 3. 分别转化为int型，相加操作
        int addRes = Integer.parseInt(sb1.toString()) + Integer.parseInt(sb2.toString());
        // 4. 转换为字符串然后反转
        String execRes = new StringBuilder((addRes + "")).reverse().toString();
        // 5. 把结果字符串转换为结果链表，返回
        char rootChar = execRes.charAt(0);
        ListNode root = new ListNode(Character.getNumericValue(rootChar));
        
        ListNode cur = root;
        for (int i = 1; i < execRes.length(); i++) {
            char c = execRes.charAt(i);
            cur.next = new ListNode(Character.getNumericValue(c));
            cur = cur.next;
        }
        
        return root;
    }
}

but...

在转换为整型后执行+操作时，没有考虑超过Integer.MAX_VALUE的情况，即使转换为double也是治标不治本，那么只能换一种思路，让两个链表节点依次相加，同时考虑进位的问题

暴力实现法2：2个链表依次节点值相加，超过10进位即可

思路：

  • 分别使用list1，list2存储l1，l2数据，分别获取对应链表长度
  • 以list1，list2中元素少的为主进行遍历，相加

  • 具体操作为：如果list1元素比list2多，则第一阶段遍历[0,list2.size())，遍历时更新list1中对应元素的值，同时获取向后进位的值，第二阶段遍历[list2.size() , list1.size())，操作与第一阶段相同，重点要注意如果最后一位相加后还有进位，则list1要新增一个元素，元素值为进位的值1

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        // 1. 分别使用arrayList存储l1,l2数据
        List<Integer> list1 = new ArrayList<>();
        while (l1 != null) {
            list1.add(l1.val);
            l1 = l1.next;
        }
        List<Integer> list2 = new ArrayList<>();
        while (l2 != null) {
            list2.add(l2.val);
            l2 = l2.next;
        }

        // 2. 以list1,list2中元素少的为主进行遍历，相加
          if (list1.size() > list2.size()) {
            int parent = 0;
            // 第一阶段
            for (int i = 0; i < list2.size(); i++) {
                int addRes;
                int cur = 0;

                if (i == 0) {
                    addRes = list2.get(i) + list1.get(i);
                } else {
                    addRes = list2.get(i) + list1.get(i) + parent;
                }
                if (addRes < 10) {
                    cur = addRes;
                    parent = 0;
                } else if (addRes == 10) {
                    parent = 1;
                } else {
                    cur = addRes % 10;
                    parent = 1;
                }

                list1.set(i, cur);
            }

            // 第二阶段
            for (int i = list2.size(); i < list1.size(); i++) {
                int addRes = list1.get(i) + parent;
                int cur = 0;

                if (addRes < 10) {
                    cur = addRes;
                    parent = 0;
                } else if (addRes == 10) {
                    parent = 1;
                } else {
                    cur = addRes % 10;
                    parent = 1;
                }

                list1.set(i, cur);
            }

            if (parent == 1) {
                list1.add(1);
            }

           ListNode root = new ListNode(list1.get(0));

           ListNode cur = root;
           for (int i = 1; i < list1.size(); i++) {
               cur.next = new ListNode(list1.get(i));
               cur = cur.next;
           }

           return root;
        } else if (list1.size() < list2.size()) {
            int parent = 0;
            for (int i = 0; i < list1.size(); i++) {
                int addRes;
                int cur = 0;

                if (i == 0) {
                    addRes = list1.get(i) + list2.get(i);
                } else {
                    addRes = list1.get(i) + list2.get(i) + parent;
                }
                if (addRes < 10) {
                    cur = addRes;
                    parent = 0;
                } else if (addRes == 10) {
                    parent = 1;
                } else {
                    cur = addRes % 10;
                    parent = 1;
                }

                list2.set(i, cur);
            }

            // 第二阶段
            for (int i = list1.size(); i < list2.size(); i++) {
                int addRes = list2.get(i) + parent;
                int cur = 0;

                if (addRes < 10) {
                    cur = addRes;
                    parent = 0;
                } else if (addRes == 10) {
                    parent = 1;
                } else {
                    cur = addRes % 10;
                    parent = 1;
                }

                list2.set(i, cur);
            }

            if (parent == 1) {
                list2.add(1);
            }

           ListNode root = new ListNode(list2.get(0));

           ListNode cur = root;
           for (int i = 1; i < list2.size(); i++) {
               cur.next = new ListNode(list2.get(i));
               cur = cur.next;
           }

           return root;
        } else {
            int parent = 0;
            for (int i = 0; i < list1.size(); i++) {
                int addRes;
                int cur = 0;

                if (i == 0) {
                    addRes = list2.get(i) + list1.get(i);
                } else {
                    addRes = list2.get(i) + list1.get(i) + parent;
                }
                if (addRes < 10) {
                    cur = addRes;
                    parent = 0;
                } else if (addRes == 10) {
                    parent = 1;
                } else {
                    cur = addRes % 10;
                    parent = 1;
                }

                list1.set(i, cur);
            }

            if (parent == 1) {
                list1.add(1);
            }

           ListNode root = new ListNode(list1.get(0));

           ListNode cur = root;
           for (int i = 1; i < list1.size(); i++) {
               cur.next = new ListNode(list1.get(i));
               cur = cur.next;
           }

           return root;
        }  
    }
}

优化

思路：

  • 上述解决方案，第一执行效率太低，第二代码过于冗余，第一、第二阶段的执行过程可以用递归代替

  • 分别获取2个链表的长度，以元素多的链表为主进行递归调用，递归调用时要带入一个进位值，第一次节点值相加时没有进位，所以进位值为0

  • 递归调用的终止条件：如果节点为空，则判断进位值是否为1，为1返回一个新建节点，节点值为1，否则返回null即可

  • 每层递归时，更新主链表（链表元素多的）对应节点的值，同时获取进位值

  • 下一次递归，判断次链表是否为空，带入进位值即可

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode newL1 = l1;
        ListNode newL2 = l2;
        // 分别获取链表的长度
        int len1 = 0;
        while (l1 != null) {
            len1++;
            l1 = l1.next;
        }
        
        int len2 = 0;
        while (l2 != null) {
            len2++;
            l2 = l2.next;
        }
        
        // 以链表长的为主进行递归调用
        if (len1 >= len2) {
            return addTwoNumbers(newL1, newL2, 0);
        } else {
            return addTwoNumbers(newL2, newL1, 0);
        }
    }
    
    private ListNode addTwoNumbers(ListNode l1, ListNode l2, int carry) {
        if (l1 == null) {
            if (carry == 1) {
                return new ListNode(carry);
            } else {
                return null;
            }
        }
        
        int addRes;
        if (l2 != null) {
            addRes = l1.val + l2.val + carry;
        } else {
            addRes = l1.val + carry;
        }
        
        if (addRes < 10) {
            l1.val = addRes;
            carry = 0;
        } else if (addRes == 10) {
            l1.val = 0;
            carry = 1;
        } else {
            l1.val = addRes % 10;
            carry = 1;
        }
        
        if (l2 != null && l2.next != null) {
            l1.next = addTwoNumbers(l1.next, l2.next, carry);
        } else {
            l1.next = addTwoNumbers(l1.next, null, carry);
        }
        
        return l1;
    }
}
